# UARTAI
创建一个通过串口访问DeepSeek的设备

## 文件说明
* UARTAI.eprj 嘉立创EDA工程文件，内有电路图和PCB Layout
* espDeepSeek.ino 固件源代码，请用Arduino IDE编译
* SERNTPC.ASM DOS时钟同步客户端源代码，请用TASM编译
* SERNTPC.EXE 编译后的DOS时钟同步客户端
